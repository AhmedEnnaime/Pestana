# Pestana

Pestana is a web application that provides a reservation system who allows to clients to reserve a room with a search bar to help them find the appropriate room. The application has other pages that gives an overview about the hotel and the services availables. Pestana gives admins a dashboard to manage the hotel with a bunch of stats and control over reservations, employees. clients and rooms.

## Deployment

To deploy this project you need to place it on your htdocs in xamp or www if you are using wamp.

## Tech Stack

**Client:** HTML, CSS, Javascript, Bootstrap, Sass

**Server:** PHP, Mysql

**Architecture:** MVC
